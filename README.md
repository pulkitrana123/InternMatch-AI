# AI Internship Portal

A basic web application for an AI internship portal built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Internship listings (Machine Learning, AI Research, NLP, Computer Vision, Data Science, Robotics, AI Ethics)
- Application form with validation
- Smooth scrolling navigation
- Fade-in animations
- Contact information

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## How to Run

Open `index.html` in a web browser.
